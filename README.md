# IRC 2021 Chatbot

Chat with the 2021 IRC building code

## 1. Download

`python3 ./downloader.py`

## 2. Markdown Conversion

`python3 ./markdown_converter.py`

## 4. Markdown Merge

`python3 ./markdown_merge.py`

## 4. OpenAI API Key

`export OPENAI_API_KEY=<your key>`

## 5. Run It

`python3 ./query.py`
